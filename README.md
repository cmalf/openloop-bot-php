## 🦾 Bot Script Features

- Login
- Multiple Accounts Connection through Proxy
- Complete All Task
- Bandwidth Share

## 🤔 How To Do

- Clone This Repo
  - ```bash
    git clone https://github.com/cmalf/openloop-bot-php.git
    ```
- Go To Folder
  - ```bash
    cd openloop-bot-php
    ```
- Configuration
- Run the script
  
  ```bash
  php main.php
  ```

## ⚙️ Configuration

- Add your Account information to the accounts.json file.
- Add proxy to proxy.txt file.
